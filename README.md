## 📖 About The Project

DigiTools Platform is a modern web application that showcases digital tools/products in a clean and interactive interface.
Users can browse products, manage their selections, and experience a smooth UI powered by React and modern frontend technologies.

---

---

## ⚙️ Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS + DaisyUI
- 💡 JavaScript (ES6+)
- 🔔 React-Toastify
- 📦 JSON (for product data)

---

---

## ✨ Features

- 🛍️ View products dynamically from JSON data
- ➕ Add & ❌ remove items from cart
- 🔄 Toggle between different data views (dynamic rendering)
- ⚡ Fast and responsive UI
- 🔔 Interactive toast notifications
